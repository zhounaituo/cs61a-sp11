## CS61A Spring 2011 Strcture and Interpretation of Computer Programs
This repository contains all the resources and books for self-studying CS61A that I've organized myself. I chose the  version of the course, because it teaches using Lisp language and provides complete open-source videos([archive](https://archive.org/details/ucberkeley-webcast-PL3E89002AA9B9879E) [YouTube](https://www.youtube.com/playlist?list=PLhMnuBfGeCDNgVzLPxF9o5UNKG1b-LFY9) [bilibili](https://www.bilibili.com/video/av40460492/)). If you need different  versions of courses, please click [here](https://web.archive.org/web/20230629231309/https://inst.eecs.berkeley.edu/~cs61a/archives.html).

## Learning Process
I will use the [Calender](http://inst.eecs.berkeley.edu/~cs61a/su11/) as my primary schedule, the [CS61A spring 2011 archive](https://people.eecs.berkeley.edu/~bh/61a-pages/) as my main learning resource. and the [Hexlet SICP](https://sicp.hexlet.io/) to track my progress in reading SICP.
